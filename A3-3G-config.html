Configuración Componente Remoto

El Componente Remoto permite al usuario configurar el puerto de comunicación del medidor, para poder realizar Llamadas (facturación, alarmas) y poder recibir llamadas (lecturas requeridas) del sistema central a través del modem celular AMI-EE-EG-A3.

Nota: si el medidor ya ha sido programado anteriormente, puede omitir este paso.

Direccionamiento de Modem Celular

Para el direccionamiento del modem celular debemos utilizar el programa EZM_MeterConfig_3G, este programa es proporcionado por EASYMETERING y sirve para configurar de una manera rapida las direcciones IP a la cual debe enviar la informacion del medidor.

Nota: Para la instalación del programa EZM_MeterConfig_3G, favor revisar el MANUAL DE INSTALACIÓN DEL SOFTWARE EZM_METERCONFIG_3G.
